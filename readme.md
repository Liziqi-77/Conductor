
# KV Event Manager


KV Event Manager is a plugin that schedules client requests by being aware of KV cache storage and node load. It is part of Mooncake Conductor.


**kv cache manager**


The KV cache manager section analyzes KV cache storage within Aibrix.

**kv event manager**


The KV event manager is an analysis of KV events in Aibrix.

**zmq**

Analysis of subscribing to VLLM Pods via ZMQ in Aibrix.

A simple version of zmq implemented in Python and Go.
